Хорошим стилем считается называть пакеты, классы, методы и переменные на английском языке,
причем не транслитом, а именно английскими словами, примеры смотрите ниже.

Именнование пакетов
Классы должны быть в пакетах. Пакеты принято именновать как название домена компании с конца.
Например: "mtuci.ru" превращается в название пакета "ru.mtuci"


Именование классов
Классы в Java принято именновать в нотации PascalCase, то есть первая буква в каждом слове
в названии пишется с большой буквы, а остальные маленькие. Название класса должно быть существительным.
Примеры именнования:
Car - класс автомобиль
ElectricCar - класс електрический автомобиль, существительное
PetrolCar - класс бензиновый автомобиль, существительное
HttpRequestListener - класс слушатель HTTP запросов, существительное


Именование методов
Методы в Java принято именновать в нотации camelCase, то есть первая первое слово пишется маленькими буквами,
а у каждого следующего первая буква большая. Название метода должно быть глаголом,
потому что метод выполняет какое-то действие.
Примеры именнования:
sum() - суммировать, глагол
subtract() - вычитать, глагол
multiply() - умножать, глагол
divide() - делить, глагол
